MeetSync — Бронирование переговорных комнат

Стек: 1С 8.3, СКД, регистры

Разработана конфигурация для учёта бронирований: справочники «Сотрудники», «Переговорные комнаты» (с табличной частью «Оснащение»), «Правила бронирования».

Создан документ «Заявка на бронирование» с автоматическим расчётом стоимости на основе цен оснащения (регистр сведений).

Разработан документ «Акт выполненных услуг» для фиксации оказания услуг.

Построены отчёты: «Календарь загрузки» (занятость комнат по дням) и «Прибыльность комнат» (регистр накопления «Бронирования» с ресурсом «Прибыль»).
